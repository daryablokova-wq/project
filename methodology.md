# Methodology

 The project followed a multi-step methodology to explore and enrich the knowledge about the cultural event “Il Tesoro dei Medici” using the ArCo Knowledge Graph and LLMs.

## the process step-by-step:

1. **Selecting the topic by querying the KG**  

2. **Exploring the ArCo’s ontology to identify classes**  

3. **Retrieving the existing information about our event from the KG**  

4. **Stating the main problem to solve during the project**  

5. **Engineering the prompts for the LLMs**  

6. **Comparing LLMs’ outputs**  

7. **Designing new triplets, thus enriching ArCo’s ontology**  

8. **Website creation and publishing**
